# Claudetor

> Claudetor notes your agent's work and syncs it to the cloud through a private MCP connection. Resume exactly where you left off—every session, every machine.

## Features

- Persistent agent memory
- Private MCP synchronization
- Cross-device continuity
- Session recovery
- Project-aware context
- Seamless agent handoffs

## Why Claudetor?

AI agents lose context. Claudetor preserves it.

```text
Work Session
     ↓
Agent Activity
     ↓
Claudetor Records Progress
     ↓
Private MCP Sync
     ↓
New Device / New Session
     ↓
Instant Resume
```

## Example

```bash
$ agent build feature

✓ Analyze repository

```

```bash

```
